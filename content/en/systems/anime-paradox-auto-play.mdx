<Section icon={Map} title="How to Configure the Native Auto Play Feature">

Setting up the native **anime paradox auto play** feature is straightforward, but maximizing its efficiency requires understanding how the AI prioritizes actions. When enabled, the game automatically handles unit placement, upgrades, and active skill triggers. However, because the system does not automatically select and start new portals, mobile players will still need to utilize a simple screen clicker to loop matches continuously.

To get the most out of this system, you should configure your settings before entering a match to ensure your units deploy immediately upon wave start.

<Steps>
  <Step num="1" title="Access Settings Menu">
    Open the in-game settings menu from the main lobby or during the preparation phase of any match.
  </Step>
  <Step num="2" title="Enable Auto Toggles">
    Locate the automation section and toggle 'Auto Play', 'Auto Spawn', 'Auto Upgrades', and 'Auto Use Skills' to the 'On' position.
  </Step>
  <Step num="3" title="Select Default Behaviors">
    Set these configurations as default so they automatically activate at the start of every subsequent match.
  </Step>
  <Step num="4" title="Deploy and Monitor">
    Enter a portal map and observe the initial unit placement to ensure your primary damage dealers are positioned correctly.
  </Step>
</Steps>

The native system offers several toggles that can be customized based on your current roster and the difficulty of the portal you are running.

| Setting Name | Function | Recommended Status |
| :--- | :--- | :--- |
| Auto Play | Activates the core automation AI | Enabled for all farming runs |
| Auto Spawn | Automatically deploys units from your bar | Enabled (Prioritizes cheapest units first) |
| Auto Upgrade | Spends gold to upgrade deployed units | Enabled (Prioritizes oldest units) |
| Auto Use Skills | Triggers active abilities instantly on cooldown | Enabled (Maximizes active DPS) |
| Force Restart | Instantly restarts the current match | Manual use (For failed runs) |
| Force End | Immediately terminates the match | Manual use (To save time) |

<Callout type="warning" title="Placement Limitations">
The native auto play AI occasionally struggles with complex map layouts. It places units based on their base range, which means units with mediocre range traits might be positioned sub-optimally. For maximum efficiency, use units with superior range traits or wide AoE attacks.
</Callout>
