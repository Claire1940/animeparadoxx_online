<Section icon={Sparkles} title="Macro Settings & TinyTask Configuration">

TinyTask is a lightweight utility that records mouse movements, clicks, and keyboard inputs, playing them back in a continuous loop. Because the tool relies entirely on coordinate-based actions, minimizing game lag and UI changes is critical. If a notification pops up or a frame drops, the macro may click the wrong area, ending the automated run prematurely.

Before starting the recording process, configure TinyTask settings to ensure reliability. The playback speed must remain at normal velocity, and the loop settings must be adjusted to repeat indefinitely.
